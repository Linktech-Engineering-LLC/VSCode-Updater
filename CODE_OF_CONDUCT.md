# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge
We pledge to make participation in this project a harassment‑free experience for everyone, regardless of experience level, technical background, or personal identity. We commit to maintaining a respectful, constructive, and professional environment.

## Our Standards
Examples of behavior that contributes to a positive environment include:
- Demonstrating empathy and respect for others
- Providing clear, actionable feedback
- Accepting responsibility for mistakes and learning from them
- Focusing on what is best for the project and its users

Examples of unacceptable behavior include:
- Harassment, insults, or derogatory comments
- Trolling or intentionally disruptive behavior
- Publishing private information without permission
- Any conduct that would be considered inappropriate in a professional setting

## Enforcement Responsibilities
Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior. They may take appropriate corrective action in response to any behavior deemed inappropriate, threatening, or harmful.

## Scope
This Code of Conduct applies within all project spaces, including GitHub issues, pull requests, discussions, and any other project‑related communication.

## Enforcement
Instances of abusive, harassing, or otherwise unacceptable behavior may be reported to the project maintainer at:

**contact@linktech.engineering**

All complaints will be reviewed and handled appropriately.
